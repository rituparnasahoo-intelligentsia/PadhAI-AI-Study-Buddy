# 🎓 PadhAI — Your AI Study Buddy

PadhAI is an AI-powered study assistant built using **NVIDIA NIM**, **Llama 3.1**, and **Gradio**.

It helps Indian students with:

* 📚 Scholarships (NSP, INSPIRE, Pragati)
* 🏆 Olympiads (IMO, NTSE, KVPY)
* 🎯 Career Guidance
* 📅 Exam Deadlines
* 💼 Internships & Hackathons

-------------------------------------------------------------------------------------------------------------

# Features

* NVIDIA NIM API Integration
* Llama 3.1 70B Model
* Clean UI with Gradio
* Quick Question Buttons
* Multi-Language Support (English + Hindi)
* Retry Logic for API Limits

----------------------------------------------------------------------------------------------------------------

# Tech Stack

* Python
* NVIDIA NIM API
* Llama 3.1 70B
* Gradio
* OpenAI SDK

--------------------------------------------------------------------------------------------------------------------------------

# Installation

```bash
pip install openai gradio
```

Run:

```bash
python padhai_nvidia.py
```
